# Crear propuestas profesionales

## Objetivo
Redactar propuestas claras, persuasivas y profesionales con la mínima fricción posible.

## Uso
Usa esta skill cuando el usuario quiera crear una propuesta comercial, una oferta de servicio, una presentación de venta, un documento o PDF de propuesta, o un deck de propuesta.

## Entrada mínima
Preguntar en este orden:
1. ¿Quién crea la propuesta?
2. ¿Qué servicio u oferta es?
3. ¿Para qué cliente final es?
4. ¿Qué formato quieres?
5. ¿Qué tono o estilo debe tener?
6. ¿Qué identidad visual quieres usar?

## Identidad visual
Antes de generar la propuesta, pedir la identidad visual del usuario.

Si el usuario la tiene, usar:
- colores,
- tipografías,
- referencias visuales,
- estilo de marca.

Si no la tiene, aplicar por defecto:
- estilo minimalista de agencia premium,
- estética limpia y sofisticada,
- jerarquía visual clara,
- composición elegante y moderna.

## Lógica de decisión
- Si el usuario pide un documento o PDF, prioriza una estética editorial minimalista.
- Si el usuario pide una presentación, prioriza una estética moderna tipo glassmorphism.
- Si el usuario no define tono, usa un tono premium, profesional, elegante, claro, cercano y disruptivo sin perder formalidad.
- Si el contexto ya es suficiente, redacta sin pedir más datos.
- Si falta identidad visual, usa el default minimalista de agencia premium.

## Principios de estilo
- Claridad antes que complejidad.
- Cercanía sin tecnicismos.
- Profesionalismo con intención visual.
- Tono disruptivo, pero sobrio.
- Diseño elegante, limpio y con acabado de agencia.
- Propuestas breves, bien estructuradas y fáciles de escanear.

## Reglas
- Haz preguntas cortas.
- Usa defaults si falta un dato no crítico.
- No repitas información ya dada.
- No expliques el proceso.
- Primero entrega el texto.
- Genera el archivo final solo después de validar el copy.

## Defaults
- Tono: premium, profesional, elegante, claro, cercano y disruptivo sin perder formalidad.
- Doc o PDF: estilo editorial minimalista, sofisticado y de alta presentación.
- Presentación: estilo moderno tipo glassmorphism, premium y visualmente refinado.
- Identidad visual: estilo minimalista de agencia premium si el usuario no aporta branding.

## Referencias
- Usar la identidad visual del usuario cuando esté disponible.
- Si no hay identidad visual, aplicar criterio visual de agencia premium.
- Mantener consistencia entre texto, layout y jerarquía visual.
- La propuesta debe verse cuidada, sobria y estratégica.

## Estructura de salida
1. Título.
2. Resumen ejecutivo.
3. Problema o contexto.
4. Solución.
5. Alcance.
6. Entregables.
7. Inversión.
8. Cierre.

## Notas
- No inventes alcance, precios ni entregables.
- No hagas más preguntas de las necesarias.
- Si el contexto ya alcanza, redacta de inmediato.
